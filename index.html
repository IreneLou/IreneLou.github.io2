Use Socket to echo GWC

Author  published on 2020-05-01   included in Note
wireshark 剪封包訊息 GWC使用UDP GTPv2傳送封包，搜尋關鍵字時要找GWC的gtpu port打“udp.port==2123”或“gtpv2”，如果搜尋"echo"會導致搜尋範圍太大沒有反應 到echo request的GTP訊息欄位，右鍵copy的選項可以選擇轉換類型，這次選hex，獲得 request msg : 40010009000002000300010046 response msg: 400200090000020003000100b8 use Socket to send ECHO Request import socket req_message = '40010009000002000300010046' #初始設定 gtpc_sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM) # 如果是UDP一來一回request & response不需要加bind gtpc_sock.bind(('0.0.0.0', Port)) # <- tuple style # socket msg type為byte gtpc_sock.sendto(bytes.fromhex(req_message), (IP, Port)) # receive from 目標IP只會收一遍，1024為buffer size try: gwc_rsp, address = gtpc_sock.recvfrom(1024) # 要判斷有沒有time out，有沒有回傳訊息 except socket.
